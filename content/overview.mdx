# OVERVIEW

Spoke is a composable system for building robots.  Robots are complex, so building them requires understanding and building many different kinds of systems.

## Body

![](images/body.png)

Robots can take many forms.  [Spoke Body](/docs/spoke-body/spoke-body) is the system that permits robots to be constructed from interchangeable parts.
